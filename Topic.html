Grade 11 Computer Science Syllabus

Welcome to the Grade 11 Computer Science E-notes. Here you can find the complete syllabus of Grade 11 Computer Science divided into 8 comprehensive chapters.

Chapter 1: Computer System

1. Introduction of Computer

  • Definition, characteristics and application of computer
  • Measurement unit of processing speed and storage unit
  • Super, Mainframe, Mini and Microcomputers
  • Mobile Computing and its Application

2. Computer System and I/O Devices

  • Concept of computer architecture and organization
  • Components of computer system
  • Microprocessor: basic concepts, clock speed, word length
  • Bus System: data bus, address bus and control bus
  • Primary memory: RAM, ROM, Cache, Buffer
  • Secondary Memory: Magnetic Disk, Flash Memory, Optical Disk
  • Input Devices – Keyboard, Mouse, Scanner, etc.
  • Output Devices: Monitor, Printer, Speaker
  • Hardware Interfaces: USB, HDMI, Expansion Slots

Chapter 2: Number System and Boolean Logic

1. Number System and Conversion

  • Decimal, Binary, Octal, Hexadecimal Number System
  • Binary addition, subtraction
  • One's and Two's complement methods

2. Logic Function and Boolean Algebra

  • Introduction to Boolean algebra
  • Boolean values, truth table, Boolean expression
  • Logic Gates – AND, OR, NOT, NAND, NOR, XOR, XNOR
  • Laws of Boolean algebra

Chapter 3: Computer Software and OS

1. Computer Software

  • Definition of software
  • Categories: System, Utility, Application, Web Apps

2. Operating System

  • Introduction to Operating System
  • Role and Functions of OS
  • OS terminology: multiprogramming, multitasking
  • Windows OS features and management
  • Open source and Mobile Operating Systems

Chapter 4: Application Package

1. Office Package

  • Word processor
  • Presentation tool
  • Spreadsheet package
  • Database management system

Chapter 5: Programming Concepts

1. Programming Concepts

  • Programming languages types
  • Compiler, Interpreter and Assembler
  • Syntax, Semantic and Runtime errors
  • Control Structures
  • Program Design tools – Algorithm, Flowchart

2. C Programming Language

  • Structure of C program
  • Data Types, Variables, Operators
  • Input/output Functions
  • Control Statements and Loops
  • Arrays and Strings

Chapter 6: Web Technology I

1. Web Development

  • Web browsers and search Engines
  • Internet & web technologies overview
  • Content Management System (CMS)

2. HTML

  • Structure of HTML
  • Basic HTML Tags and Attributes
  • Lists, Links, Tables, Forms
  • HTML5 Elements and Graphics

3. CSS

  • Introduction to CSS
  • Inline, Embedded and External CSS

Chapter 7: Multimedia

  • Introduction to Multimedia
  • Components: Text, Graphics, Audio, Video, Animation
  • Application of Multimedia

Chapter 8: Information Security

  • Digital society and computer ethics
  • Concept of Information security
  • Cybercrime and Malicious software
  • Protection from cybercrime
  • Intellectual Property Right
  • Cyber Law in Nepal
  • ICT Policy in Nepal
