Chapter 6: Web Technology - I

Introduction to the Internet

The Internet is the global network of networks, a huge collection of computers all over the world that is inter-connected to one another in various ways to form a web-net.

Components required for Internet Connections

The basic requirements for the Internet are:

  • A TCP/IP (Transmission Control Protocol/Internet Protocol) enabled computer with a web browser.
  • An account with an ISP (Internet Service Provider)
  • A telephone line
  • A MODEM to connect to the telephone line and computer
  • A set of computer hardware with the following capacity:
    • Minimum 386 microprocessor chip with 16 MB RAM
    • Color monitor with 256 color support and 640x480 resolution
    • A hard disk of 200 MB free space
    • Multimedia sound card and speaker

Internet Services and Applications

  • World Wide Web (WWW): The WWW is a series of services that are interconnected through hypertext, allowing links in web documents. Created in 1990 at the European Particle Physics Laboratory in Geneva, Switzerland, it was designed to exchange and share data over the Internet using the protocol known as Hypertext Transfer Protocol (HTTP).
  • Web Browser: A web browser is a computer program that accesses web pages and displays them on a computer screen. It allows users to interact with web pages. Examples include Internet Explorer, Safari, Mozilla Firefox, Opera, Depnet Explorer, Netscape Navigator, and Mosaic.
  • FTP - File Transfer Protocol: It allows users to move or transfer files between computers on the Internet. These files can include data, graphics, text, and more.
  • Electronic Mail (E-mail): E-mail refers to sending and receiving messages electronically through the Internet. It can include text, graphics, video, audio, and other forms of digital content.
  • Voicemail: Voicemail is a service that records audio messages when telephone calls are unanswered. It provides a way to communicate with people by leaving messages.
  • USENET – View and News: USENET is a Bulletin Board Service (BBS) for the Internet, organized into newsgroups which cover various topics.
  • Chat: Chat allows users to communicate with each other in real-time via text. Users can join chat rooms to connect with others who are online.
  • IRC - Internet Relay Chat: IRC is a multi-user live chat facility, where users can create channels and communicate with others in real-time.
  • Video Conferencing: This technology allows face-to-face communication using sound and video, enabling real-time visual communication across networks.
  • Universal Resource Locator (URL): A URL is the address that identifies the location of a resource on the Internet, using a format that includes a protocol (e.g., http), host computer name (e.g., www), domain name (e.g., animalplanet), and domain type (e.g., com).
  • E-Fax: E-Fax allows sending and receiving faxes over the Internet, similar to E-mail.
  • TELNET – Remote Login: Telnet enables users to login to a remote host computer and work on it as if they were using that computer locally.
  • Navigators – Information Servers: These help users discover, locate, and retrieve information across various objects on the Internet. Examples include Gopher, Archie, and Hytelnet.

Internet Terminologies

  • ISP: Internet Service Provider, an institution that provides access to the Internet. Examples include Mercantile, Everest net, World link in Nepal.
  • Link: A link is a shortcut to another webpage on the Internet. Links are usually displayed in a different color from the surrounding text.
  • Search Engine: A search engine is software that searches the web for websites. Examples include Yahoo, Excite, and AltaVista.
  • Domain Name and IP Address: A domain name identifies a site on the Internet. The IP address is the numeric address associated with it, used to transfer data between computers.
  • DNS Server: DNS (Domain Name Service) Server is a system that converts domain names into IP addresses for routing over the Internet.
  • Web Server: A web server stores the webpages and responds to requests for those resources from web browsers.
  • Usernet Newsgroups: These are specialized headlines or newsgroups on the Internet for specific topics like science or recreation.
  • Mailing Lists: A mailing list is a collection of email addresses that can be used to send messages to multiple recipients at once.
  • POP Server: The Post Office Protocol server stores incoming emails for retrieval by the user.
  • SMTP Server: Simple Mail Transfer Protocol (SMTP) is used to transfer electronic mail between computers.
  • FTP: File Transfer Protocol, a set of rules used to transfer data between computers on the Internet.
  • Anonymous FTP: A method for accessing archived sites with software, documents, and other files, often used without requiring a user account.
  • HTTP (Hypertext Transfer Protocol): A protocol used by web browsers to communicate with web servers to transfer HTML documents.
  • Proxy Server: A proxy server is used to improve browser speed and update information on websites.
  • Internet Roaming: This allows users to access their Internet accounts from anywhere in the world if the ISP is part of a global roaming community.

Introduction to HTML

HTML stands for Hypertext Markup Language, used to create web pages. It is a text-based language that includes various tags and attributes to design web pages. HTML was developed by Tim Berners-Lee based on SGML (Standard Generalized Markup Language). The web browser interprets these tags to display content on the page. The versions of HTML include HTML 1.0, HTML 2.0, HTML 3.2, HTML 4.01, and HTML 5.0.

Use of HTML

  • Create the basic layout and design of web pages.
  • Enable embedding of images, videos, and audio.
  • Format text and customize elements like bold, italic, and underline.
  • Create links and lists for navigation.

Advantages of HTML

  • Easy to use and flexible.
  • Supported by all web browsers and operating systems.
  • Allows quick and easy modifications of content.
  • Compatible with search engines.
  • Fast to load.

Disadvantages of HTML

  • Cannot create dynamic webpages by itself (requires languages like PHP, JavaScript, etc.).
  • Limited interactivity and aesthetics without additional scripting.
  • Incompatibilities across browsers.

HTML Tags

HTML tags are commands that the browser uses to organize and display content. Tags are written within angular brackets <> and can be either pair tags (with opening and closing tags) or singular tags (with only an opening tag).

Attributes

Attributes provide additional information about an HTML element and allow customization. For example, the bgcolor attribute can set the background color of a webpage.

Steps to Write and Execute HTML Document

  1. Write the HTML content in a text editor.
  2. Save the file with a .html extension (e.g., sample.html).
  3. Open the file in a web browser to view the webpage.

Common Web Design Terminologies

  • Webpage: An HTML document that displays content on the web.
  • Website: A collection of related webpages connected via hyperlinks.
  • Home Page: The main page of a website that loads first.
  • Web Browser: A program used to access and display web pages. Examples include Google Chrome, Safari, Mozilla Firefox, etc.
  • URL: Uniform Resource Locator, the address used to access a webpage.

DBMS (Database Management System)

Data are raw facts, and information is processed data that provides meaningful results. A database organizes data so it can be retrieved easily when needed. A DBMS is a software system that manages databases, making it easier to store, process, and access data. Examples include MS-Access, MySQL, and Oracle.

Advantages of DBMS

  • Quick and easy data retrieval.
  • Reduced data redundancy.
  • Improved data security and privacy.
  • Allows data sharing across networks.
  • Provides organized data storage.

Disadvantages of DBMS

  • High installation and operating costs.
  • Requires significant disk space.
  • May have security and privacy concerns.
  • Needs skilled personnel to maintain and operate.

Questions and Answers

1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure of a webpage using elements and tags.

Example:

Hello, World!


This is a paragraph.

2. What are basic HTML tags?

Common HTML tags include:

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a href="URL">: Links
  • <img src="image.jpg">: Images
  • <ul> & <ol>: Lists
  • <table>: Tables

3. What is CSS?

CSS (Cascading Style Sheets) is used to style HTML elements and make webpages visually appealing.

Example:

p {
    color: blue;
    font-size: 18px;
}

4. What are basic CSS properties?

Common CSS properties include:

  • color: Changes text color
  • background-color: Sets background color
  • font-size: Adjusts text size
  • margin: Controls space outside an element
